## 소스별 데이터 항목 상세

### 1. KRX 한국거래소
접근: Playwright 브라우저 자동화 (현재 구현됨)

| 분류 | 데이터 항목 |
|------|------------|
| ETF | 전종목 시세 (NAV/상장주수/순자산총액/거래량), 순설정/순환매 |
| 수급 | 외국인/기관 순매수 상위종목, 투자자별 매매동향 (13개 구분) |
| 주식 | 전종목 OHLCV, 상장종목 기본정보 (업종/시총/상장주식수) |
| 지수 | KOSPI/KOSDAQ/KONEX, 업종별 지수 |
| 파생 | 선물/옵션 시세 및 미결제약정, 파생상품 투자자별 매매동향 |
| 채권 | 채권 시세 및 수익률 |
| 공매도 | 잔고/거래량/거래대금 (종목별/일별) |
| 기타 | 배당/권리락, 프로그램 매매 (차익/비차익) |

### 2. 한국은행 ECOS
접근: 공개 REST API, API 키 필요 (ecos.bok.or.kr 에서 무료 발급)

| 분류 | 데이터 항목 |
|------|------------|
| 금리 | 기준금리, 국고채 (1y~50y), 콜금리, CD금리(91일), CP금리 |
| 환율 | 원/달러, 원/엔, 원/유로, 원/위안 등 일별 환율 |
| 통화/신용 | 통화량 (M1/M2/Lf/L), 가계신용, 외환보유액 |
| 국제수지 | 경상수지, 외국인 증권투자 잔액 |
| 거시경제 | GDP, CPI, PPI, 수출입 물가지수, 산업생산지수 |
| 심리지표 | BSI (기업경기실사), CCSI (소비자심리) |

### 3. 네이버금융 ⭐ 우선 소스
접근: 비공개 API (스크래핑), 로그인 불필요
**방향: KRX 다음으로 네이버금융 우선 사용. 로그인 불필요하고 데이터 범위가 넓음.**

| 분류 | 데이터 항목 |
|------|------------|
| 주식 시세 | OHLCV (분봉/일봉/주봉/월봉), 시가총액 |
| 지수 | KOSPI/KOSDAQ/KRX100/KRX300 |
| 스크리닝 | 상한/하한가, 52주 신고/신저, 거래량 상위 |
| 해외/원자재 | 다우/S&P500/나스닥/닛케이/항셍, 금/은/WTI/구리 |
| 업종/테마 | 업종별 등락률, 테마별 등락률 |
| 기업정보 | 재무제표 요약, PER/PBR/EPS/BPS |
| 공시 | 뉴스, 공시 링크 |

### 4. 금융투자협회 FreeSIS
접근: 비공개 API, 일부 로그인 필요

| 분류 | 데이터 항목 |
|------|------------|
| 채권 금리 | 시가평가 수익률 (AAA~C 등급별, 만기 1y~30y) — **국내 유일 무료 소스** |
| 스프레드 | 회사채 신용 스프레드 (AAA/AA/A/BBB - 국고채) |
| 단기 금리 | CP(91일), CD(91일), RP(1일/7일), 콜금리 |
| 펀드 | 설정원본/순자산, 유형별 자금 유출입 (주식/채권/혼합/MMF) |
| **수급지표** | **투자자예탁금, 신용융자 잔고, 신용대주 잔고, 미수금** |
| 발행시장 | 채권 발행 통계 (국채/회사채/금융채/ABS/MBS) |

### 5. 금융감독원 DART ⭐ 우선 소스
접근: 공개 REST API, API 키 무료 발급.
**방향: 공시/재무/지분 데이터는 DART 우선 사용. 공식 REST API라 안정적.**

| 분류 | 데이터 항목 |
|------|------------|
| 공시 | 사업/반기/분기/주요사항보고서 전체 |
| 지분 | 임원/주요주주 지분변동, 5% 대량보유 신고 |
| 재무제표 | 손익계산서/대차대조표/현금흐름표 (단일/연결) |
| 지표 | EPS, BPS, DPS, PER, PBR |
| 자본 변동 | 유상증자/감자/자사주/CB/BW/RCPS 발행 |

### 6. 통계청 KOSIS
접근: 공개 REST API, API 키 무료 발급. 주식 데이터 없음.

| 분류 | 데이터 항목 |
|------|------------|
| 물가 | CPI (품목별/지역별), PPI |
| 무역 | 수출입 (HS코드 품목별, 국가별) |
| 생산/소비 | 광공업/서비스업 생산지수, 소매판매액지수 |
| 고용 | 고용률, 실업률, 취업자수 (산업별) |
| 경기지수 | 선행/동행/후행 CI/DI |

### 7. 한국예탁결제원 SEIBro
접근: 웹스크래핑, 일부 로그인 필요

| 분류 | 데이터 항목 |
|------|------------|
| 외국인 보유 | 종목별 보유 주식수/금액/비중, 국가별 분류 |
| **보유한도** | **외국인 보유한도 및 한도소진율** (100% 근접 시 추가 매수 불가 신호) |
| 채권 | 외국인 국내채권 보유 잔액 |
| 대차/공매도 | 대차잔고, 종목별 공매도 잔고 |

### 8. 한국투자증권 KIS API
접근: REST + WebSocket, 계좌 개설 후 API 키 발급 (무료)

| 분류 | 데이터 항목 |
|------|------------|
| 국내 시세 | 현재가, 호가 10단계, 체결(틱), OHLCV |
| 수급 | 투자자별 매매동향 실시간, 프로그램 매매 |
| 해외 | 미국/일본/중국/홍콩/베트남 주식 |
| **실시간** | **WebSocket 스트리밍 (동시 41개 스트림)** |

### 9. FRED (St. Louis Fed)
접근: 공개 REST API, API 키 무료. `pip install fredapi`

| 시리즈 ID | 데이터 항목 |
|-----------|------------|
| DFF | 미국 기준금리 |
| GS2/GS10/GS30 | 미국 국채 2y/10y/30y |
| T10Y2Y | 장단기 스프레드 (경기침체 선행지표) |
| DTWEXBGS | 달러 인덱스 (Broad) |
| VIXCLS | VIX 지수 |
| DEXKOUS | 원/달러 환율 |
| CPIAUCSL/UNRATE | 미국 CPI, 실업률 |

### 10. Investing.com
접근: 비공개 API, `investiny` 라이브러리

특이사항: **경제 캘린더 (FOMC/CPI/NFP 발표일, 예상치/실제치)** 가 이벤트 드리븐 전략에 유용.
